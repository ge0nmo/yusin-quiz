== 학습 API

- `POST /api/v1/study/finish` 는 counts 기반 요약을 authoritative 계약으로 사용한다.
- `finalScore` 는 기존 앱 하위호환을 위한 deprecated 필드이며 현재 `correctCount` 와 같은 값을 유지한다.
- 학습 로그는 로그인 사용자 기준으로 `practice`, `real/exam` 모두 포함한다.
- `practice` 는 세션 내 같은 문제의 첫 제출만 집계하며, 같은 세션에서 답을 바꿔도 추가 집계하지 않는다.
- 이 집계는 하루/전체 기간 unique 문제 수가 아니라 세션 시도 누적값이다.
- `POST /api/v1/study/exam/start` 의 `submittedAnswers[]` item 은 `problemId`, `choiceId`, `isCorrect(boolean)` 을 모두 반환한다.
